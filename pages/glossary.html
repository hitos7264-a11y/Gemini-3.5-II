用語集
— Gemini 3.5関連キーワード

本サイトで頻出する略語・専門用語・コードネームを整理。アルファベット順／五十音順を混在させ、「すぐに引ける」設計にしています。

A B C D E F G H I L M N P R S T V

A

Antigravity
アンチグラビティ
Googleのエージェンティック開発プラットフォーム。コード生成・実行・デバッグを統合的に扱う。Gemini 3/3.1と深く結合しており、3.5世代ではAntigravity 2がリリースされると予想される。
ARC‑AGI‑2
アーク エージーアイ ツー
François Cholletが提唱した抽象的一般知能ベンチマーク。パズル様の課題で人間には直感的だがLLMが苦手とされる。Gemini 3.1 Proは77.1%、3.5では82-85%が予想される。
Axion CPU
アクシオン
Googleが開発するArm系の自社CPU。Ironwood TPUと組み合わせてAI Hypercomputerを構成する。

B

BrowseComp
ブラウズコンプ
ブラウザ操作を通じた情報収集・タスク実行を評価するエージェンティック検索ベンチマーク。Gemini 3.1 Proは85.9%で首位級。

C

C2PA
シー ツー ピーエー
Coalition for Content Provenance and Authenticityの略。コンテンツの出所証明標準。AI生成物にメタデータで制作情報を付与する業界標準。
Claude Opus 4.5 / 4.6
クロード オーパス
AnthropicのフラッグシップLLM。コーディング系ベンチマーク（SWE-Bench、Terminal-Bench）で強く、Gemini 3.5の主要競合。
Context Window
コンテキストウィンドウ
モデルが一度に処理できる最大トークン数。3.1 Proは入力1M/出力64K。3.5では入力2Mへの拡張が予想される。

D

Deep Think
ディープシンク
Geminiの「じっくり考える」上位モード。数分〜数十分単位のチェーン思考を回し、ARC-AGI-2で45.1%などのスコアを出す。3.5では標準搭載が濃厚。
Demis Hassabis
デミス・ハサビス
Google DeepMind CEO。2026年4月16日Fortuneインタビューで「商業競争よりAGI安全」を強調。3.5リリース時のトーン設定に大きく影響。
DN9 / D13
ディーエヌナイン / ディーサーティーン
Community.aifire.coで観測されたSnow Bunny関連の内部コード。Fierce/Ghost Falconとの関係は未確定。

E

Elo
イロ
チェスなどの対戦レーティングシステム。LLM評価ではLMArenaで使われる。基準1000、フロンティアモデルは1400-1500、Gemini 3 Proは1501で史上初の四桁Elo超え。
Enterprise SKU
エンタープライズ SKU
企業向け契約製品ライン。3.5では3-5Mコンテキストプレビュー枠などが提供されると予想される。

F

Fierce Falcon
フィアース ファルコン
LM Arenaに現れた匿名モデル。精密さ・論理・コーディング重視と解釈される（Julian Goldie等の説）。Gemini 3.5の内部エキスパートの一人？
Flash / Flash-Lite / Flash Live / Flash TTS
フラッシュ
Geminiの軽量モデル系統。Flashは汎用、Liteはコスト最適化、Liveはリアルタイム音声、TTSは音声合成。3.5でも同じサブブランドが継承される見込み。
Flex / Priority
フレックス / プライオリティ
2026年4月1日にAPIに導入された推論ティア。Flex=安価・遅延許容、Priority=優先・低遅延。Batch/Cachingとともに4ティア構造を形成。

G

Gemini Enterprise
ジェミニ エンタープライズ
Google CloudのAI機能を企業向けに統合的に提供するプラットフォーム。Vertex AIと並ぶ配信チャネル。
Gemma 4
ジェンマ フォー
Googleのオープンウェイトモデル。2026年4月2日に26B-a4b-it（MoE）と31B-it（密）が刷新公開。
Ghost Falcon
ゴースト ファルコン
Fierce Falconと対になる匿名モデル。創造性・ビジュアル・デザイン重視と解釈される説と、Gemini 3 Flash/Pro GAチェックポイント説が対立。
GPQA Diamond
ジーピーキューエー ダイヤモンド
Google Proof Q&Aの難問サブセット。博士レベルの科学QA。Gemini 3.1 Proは94.3%で人間博士レベルに接近。
GPT-5.2 / 5.4
ジーピーティー
OpenAIのフラッグシップLLM。AIMEほぼ100%、BenchLM総合84点で首位級。Gemini 3.5の最大のライバル。

H

HLE (Humanity's Last Exam)
エイチエルイー / ヒューマニティーズ ラスト エグザム
最難関の学術推論ベンチマーク。2025年にセンセーショナルに登場した「人類最終試験」。Gemini 3.1 Pro 44.4%、3.5予想48-52%。

I

Ironwood TPU (TPUv7)
アイアンウッド ティーピーユー
Googleの第7世代テンソル処理ユニット。推論特化設計。1ポッド9216チップ、42.5 ExaFLOPS(FP8)、TPU v5比10倍ピーク性能、4倍ワット性能。Gemini 3の100% TPU学習を支えた。

L

LaMarina / LM Arena
ラマリナ / エルエム アリーナ
匿名LLM同士がブラインドでバトルし、ユーザー投票でEloレーティングが決まるリーダーボード。Gemini 3 Pro、Snow Bunny、Fierce/Ghost Falconなどのステルステスト場。
LiveCodeBench Pro
ライブコードベンチ プロ
競技プログラミング形式のコードベンチマーク。Elo表示。Gemini 3.1 Proは2887 Eloで首位。3.5では3000 Elo超が予想される。
Lyria 3
リリア スリー
Googleの音楽生成モデル。2026年3月25日にClip版とPro版がリリース。全出力にSynthID透かし。

M

MCP Atlas
エムシーピー アトラス
Model Context Protocolによる多段ツール連携ワークフローの評価ベンチマーク。3.5では75-80%が予想レンジ。
MMMU-Pro
エムエムエムユー プロ
Massive Multi-discipline Multimodal Understandingの難易度向上版。画像+テキストの多分野理解。Gemini 3 Proは81%でGPT-5.1(76%)/Opus 4.5(72%)を上回る。
MoE (Mixture of Experts)
ミックスチャー オブ エキスパーツ
複数の専門家ネットワークを持ち、入力に応じて一部のみを活性化するアーキテクチャ。Gemini 1.5以降の基盤設計。3.5ではタスク別エキスパートルーティングがさらに発展する見込み。
MRCR v2
エムアールシーアール ブイツー
長文読解・検索の実効コンテキストベンチマーク。Gemini 3.1 Proは128kで84.9%だが、1Mでは26.3%に落ちる。3.5の改善ポイント。

N

NIAH (Needle in a Haystack)
ニードル イン ア ヘイスタック
長文コンテキスト評価の古典手法。干し草の山（長文）の中から針（重要情報）を正確に取り出せるかをテスト。
NotebookLM
ノートブックエルエム
Googleの文書理解・ノート生成ツール。Geminiを搭載したRAG的体験。3.5世代でさらに強化される配信チャネルの一つ。

P

Personal Intelligence
パーソナル インテリジェンス
Gemini AppがGmail、Calendar、Drive、YouTube、Maps等と連携する個人データ統合機能。2026年3月から無料開放。3.5時代はこの基盤の上でさらに深くユーザーに食い込む。
Polymarket / Manifold
ポリマーケット / マニフォールド
予測市場プラットフォーム。Gemini 3.5リリース時期などを参加者がオッズ形式で取引する。Polymarketのボリュームが87万ドル超のGemini 3.5市場は流動性豊富。

R

Robotics-ER 1.6 / 2.0
ロボティクス イーアール
Geminiの身体化推論(Embodied Reasoning)モデル。2026年4月14日に1.6へアップグレードし、空間推論・多視点理解・計器読解を強化。柔軟なthinking budgetパラメータが3.5本体と共通化される見込み。

S

Shoreline Amphitheatre
ショアライン アンフィシアター
カリフォルニア州マウンテンビューの野外音楽堂。Google I/Oの開催地として定着。2026年5月19-20日にI/O 2026が開催される。
Snow Bunny / Snowbunny
スノーバニー
Gemini 3.5の初期チェックポイントと目されるコードネーム。2025年12月〜2026年1月にGoogle AI Studioの内部パラメータやLM Arenaで観測。3000行コード一ショット生成などの伝説的デモを残した。
Sonnet 5 / Opus 5
ソネット / オーパス
Anthropicの次世代Claude。2026年半ばに登場予想。Gemini 3.5との競合軸になる可能性。
SynthID
シンスアイディー
Google DeepMindが開発したAI生成コンテンツへの透かし技術。テキスト・画像・音声・動画すべてに適用可能。Lyria 3、Flash TTS、Veoなどの出力に標準埋め込み。
SWE-Bench Verified
エスダブリューイー ベンチ
実世界のOSSリポジトリバグ修正ベンチマークのうち、人手で検証された高信頼サブセット。Gemini 3.1 Pro 80.6%、Opus 4.6は82%付近と推定。

T

Terminal-Bench 2.0
ターミナルベンチ
ターミナル（CLI）操作を通じた多段タスク遂行の評価ベンチマーク。Gemini 3.1 Pro 68.5%、Claude Opus 4.6が~70%。
Thinking Budget
シンキング バジェット
モデルが1回の推論で消費する思考量（計算コスト）の予算。Robotics-ER 1.6で柔軟な設定が導入、3.5では本体と共通化される予想。
Three-Tier Thinking
スリーティア シンキング
Low/Medium/Highの3段階で思考深度を切り替える仕組み。3.1 Proで採用。Highは実質「Deep Think Mini」。3.5ではこの段階がさらに上にシフトすると予想。
TPU (Tensor Processing Unit)
ティーピーユー
Google独自開発のAI専用プロセッサ。第7世代がIronwood。Nvidia GPU非依存のGemini学習・推論を支える。

V

Veo 3.1 / Veo 4
ヴィオ
Googleの動画生成モデル。Lite版含めエコシステムを形成。3.5世代ではVeo 4がリリース予想、長尺・音声付き生成対応。
Vertex AI
バーテックス エーアイ
Google CloudのML/AI統合プラットフォーム。Gemini APIと並ぶ企業向け配信チャネル。
