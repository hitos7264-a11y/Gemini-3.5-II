CHAPTER 01

いまGoogleのモデル棚に並んでいるもの
— 2026年4月の現在地

Gemini 3.5を予想する前に、まずは2026年4月時点で既にGoogleが出荷しているモデル群を棚卸しする。ここを押さえずに「次」を語るのは足場なしで飛ぼうとするようなものだ。3.1 Proを中核に、Flash/Live/Lite/TTS/Robotics/Embedding/Gemma/Veo/Lyriaが縦横に肥大化している様子を見ていこう。

📖 約 5 分 🎯 現状整理 🟢 高確度（公式ソース）

フラッグシップ：Gemini 3.1 Pro Preview

予想の土台として、まず現時点でGoogleが出荷しているGeminiファミリーを改めて整理しておきたい。フラッグシップは2月19日に登場したGemini 3.1 Pro Preview。これはGemini 3 Proをベースに、DeepThink系の推論力を本体に埋め込む形で生まれた「スリーティア・シンキングシステム」を備えるモデルだ。

🧠スリーティア・シンキングシステム

Low / Medium / High の3段階で思考量（計算コスト）を切り替えられる。特にHighに振り切ると実質的に「Deep Think Miniモード」として、分単位の長考を許容する構造になっている。

配信面では、VertexAI / Gemini Enterprise / Gemini API / Google AI Studio / Gemini CLI / Google Antigravity / Android Studio、さらにはGemini App / NotebookLMなどあらゆるエンドポイントに展開されており、消費者ルートでも AI Pro (月額19.99ドル) と AI Ultra (月額249.99ドル、初回3ヶ月は124.99ドルのプロモ) の両プランで上限を引き上げて利用できる。

観察：3.5の空白を埋める3.1エコシステム

このリストを眺めるだけで、Googleが「3.1」のブランドを軸に、Flash / Flash Live / Flash‑Lite / Flash TTS / Robotics / Embedding と、縦にも横にもパフォーマンスを積み増していることが分かる。「3.5の空白」を埋めるように、3.1のエコシステムが肥大化しているのだ。裏を返せば、3.5の登場は「まもなく」という空気が濃い。
